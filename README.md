
# 🎓 Tkinter ile Öğrenci Kayıt Sistemi

Bu proje, Python'da GUI (Grafiksel Arayüz) programlama için `tkinter` ve veritabanı işlemleri için `sqlite3` kullanılarak geliştirilmiş bir öğrenci kayıt sistemidir.

## 🔧 Özellikler
- Öğrenci verilerini (isim, numara, not, yaş) kaydetme
- Kayıtlı öğrenci verilerini arama
- Öğrenci verilerini güncelleme
- Kullanıcı dostu grafik arayüz
- Veriler `ogrenciler.db` adlı SQLite veritabanına kaydedilir

## 📁 Kullanılan Teknolojiler
- Python
- Tkinter (GUI)
- SQLite (Veritabanı)
- messagebox (Bilgi uyarı kutuları)

## 🧪 Nasıl Kullanılır?
1. Python 3 yüklü olmalıdır.
2. `ogrenci_kayit.py` dosyasını çalıştırın.
3. Arayüz üzerinden veri girişlerini yaparak sistemi test edin.
